## 💾 Instalación y ejecución

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tuusuario/portfolio.git
   cd portfolio
   ```
2. Instala dependencias con npm o pnpm:
   ```bash
   npm install
   # o
   pnpm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   # o
   pnpm dev
   ```
4. Abre http://localhost:5173 (o el puerto que indique) en tu navegador.
